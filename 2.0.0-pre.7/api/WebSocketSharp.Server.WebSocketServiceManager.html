    public void AddService<TBehavior>(string path, Action<TBehavior> initializer)
    where TBehavior : WebSocketBehavior, new()
    Parameters
    Type Name Description
    System.String path

    A System.String that represents an absolute path to the service to add.

    / is trimmed from the end of the string if present.

    System.Action<TBehavior> initializer

    An Action<TBehavior> delegate or null if not needed.

    The delegate invokes the method called when initializing a new session instance for the service.

    Type Parameters
    Name Description
    TBehavior

    The type of the behavior for the service.

    It must inherit the WebSocketBehavior class.

    And also, it must have a public parameterless constructor.

    path is null.
    System.ArgumentException

    path is empty.

    -or-

    path is not an absolute path.

    -or-

    path includes either or both query and fragment components.

    -or-

    path is already in use.
